Programm ausführen:
Windows:
StickManAdventure.Desktop.exe einfach öffnen.
Linux:
StickManAdventure.Desktop.exe mit mono starten.

Level:
In dem Unterverzeichnis Levels befindet sich die JSON Datei, Level1.json. In dieser Datei können die Spieler festgelegt und die Map gebaut werden. Die Map besteht aus einen 2 Dimensionalen int Array. Jede Dimension stellt eine Layer der Map dar. 
Jeder Block hat eine ID:
Erde: 1
Grass: 2
Falle: 3
Spawnpoint: 10
Finish: 20
